About Graph Classification

Graph classification is a machine learning task where the goal is to predict the label or category of entire graphs based on their structure, node features, and edge features. This is essential for understanding graph-level properties and patterns.

Applications:
  • Molecular property prediction
  • Social network analysis
  • Protein function classification
  • Code analysis and classification
Key Features:
  • Graph-level prediction
  • Utilizes graph structure and features
  • Supports various pooling methods
  • Handles variable-sized graphs
