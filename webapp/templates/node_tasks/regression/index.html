About Node Regression

Node regression tasks involve predicting continuous values for nodes in a graph. This is useful for applications like predicting node properties, scores, or continuous attributes based on the graph structure and node features.

Applications:
  • Predicting node scores or ratings
  • Estimating continuous node properties
  • Regression on graph-structured data
  • Value prediction in social networks
Key Features:
  • Continuous value prediction
  • Graph-aware feature learning
  • Multiple regression metrics
  • Scalable to large graphs

Inductive framework for learning node representations by sampling and aggregating neighbor features.
