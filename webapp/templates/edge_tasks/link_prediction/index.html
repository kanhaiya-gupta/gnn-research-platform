About Link Prediction

Link prediction is the task of predicting missing or future connections (edges) between nodes in a graph, widely used in social networks, recommendation systems, and biological networks.

Applications:
  • Social network friend recommendations
  • Knowledge graph completion
  • Biological interaction prediction
  • Recommender systems
Key Features:
  • Edge existence prediction
  • Utilizes node and edge features
  • Supports various GNN architectures
  • Handles dynamic and static graphs
